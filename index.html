About

This project uses YOLO deep learning model to detect traffic violations such as no-helmet, illegal parking, red-light running, etc.

Features

YOLO Detection

High-speed and high-precision identification.

Multi-violation

Helmet / Red-light / Illegal parking.

ROS2 Ready

Can be deployed on robot / inspection car.
